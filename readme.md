###FloppyDiskUtility

Написано в студенческие времена на MASM (или TASM, уже не помню) just for fun. Работает под DOS, и, кажется, под Windows 9x.<br />
Умеет сохранять образ дискетки в файл и обратно.<br />
В history.txt оригинальные комментарии тех времён (монологи сумасшедшего).<br />
<s>Пулл-реквесты принимаются.</s>
